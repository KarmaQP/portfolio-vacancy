# Personal Portfolio (Vue.js)

Личное интерактивное веб-портфолио, разработанное с нуля на Vue.js.  
Проект демонстрирует информацию обо мне, ключевые навыки, примеры работ и контактные данные.  
Особый акцент сделан на UI/UX, анимации, адаптивную верстку и атмосферный визуальный стиль в духе ретро / аниме 80-х.

---

## Features

- SPA-приложение на Vue 3
- Адаптивная верстка (desktop, laptop, tablet, mobile)
- Карточная структура секций
- Плавный скроллинг по якорям
- Интерактивная навигация
- Анимации с использованием GSAP
- Hover-анимации
- Модальные окна с подробным описанием проектов
- Затемнение фона и закрытие модальных окон по клику
- Современный UI с неоновыми акцентами

---

## Tech Stack

### Frontend

- **Vue 3**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **GSAP** (анимации)

### UI / UX

- Адаптивная верстка
- Flexbox / CSS Grid
